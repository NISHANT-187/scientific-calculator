# Scientific Calculator

A modern scientific calculator built with HTML, CSS, and JavaScript. It keeps the same glass-style aesthetic as the normal calculator while adding advanced scientific functions, animated interactions, keyboard support, and click sounds.

## Features

- Aesthetic glassmorphism design
- Smooth hover, press, ripple, and entrance animations
- Click sound effect with an on/off toggle
- Degree and radian angle modes
- Scientific functions: `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `sqrt`, `log`, `ln`, `exp`, and factorial
- Constants: `pi`, `e`, and previous answer `Ans`
- Supports powers, percentages, modulo, negative values, decimals, and nested parentheses
- Responsive layout for desktop and mobile screens
- Keyboard support for numbers, operators, parentheses, decimal point, clear, delete, and calculate

## Tech Stack

- CSS
- JavaScript

## How to Use

1. Open `index.html` in your browser.
2. Use the calculator buttons or your keyboard.
3. Toggle `DEG` / `RAD` depending on the angle mode you need.

## Example Calculations

- `sin(30)` in DEG mode gives `0.5`
- `sqrt(144)` gives `12`
- `2^10` gives `1024`
- `5!` gives `120`
- `log(1000)` gives `3`
- `(2+3)*4` gives `20`
